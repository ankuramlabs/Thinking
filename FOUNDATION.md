# FOUNDATION

## Mission
Thinking is the reference specification for an AI-native Knowledge Operating System.

## Scope
This repository defines principles, architecture, and specifications. It is not an implementation repository.

## Core Principles
1. Preserve meaning through stable semantics.
2. Separate knowledge from representation.
3. Prefer architecture over implementation.
4. Maintain implementation independence.
5. Evolve through explicit architectural decisions.

## Governance
This document is the constitutional foundation of the repository.

## Normative Hierarchy
FOUNDATION.md > ADRs > Specifications > Reference Material > Implementations

## Amendment
Changes to this document require explicit architectural review and should be rare.
